Project Portfolio

SpaceMath!

This project was geared towards developing a first and second grader based math game completed during my Software Engineering course at Westminster College. From this, I really gained exposure as to what working in the industry as a Software Engineer would be like. From day one, we were thrown right into our groups and the project! It was a team effort, which I had the opportunity to lead. Completing this project gave me the confidence to pursue a Software Engineer Internship that I began in May 2019. This project was successful in teaching me the concepts and techniques of software design.
